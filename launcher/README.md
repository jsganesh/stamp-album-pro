# Launcher Files

These files let you start StampAlbum Pro without typing in a terminal.

## macOS — `.command` file
Double-click `stamp-album.command` → Terminal opens and runs the server.

## macOS — `.app` bundle (recommended)
Run `bash launcher/make-app.sh` to create `dist/StampAlbum.app`.
Open the app or drag it to `/Applications/`. No Terminal window visible.

## Windows
Double-click `stamp-album.bat`.

## Linux
Copy `stamp-album.desktop` to `~/.local/share/applications/`.
