DSL Command Reference
Document
ALBUM_TITLE("name")Set album title
ALBUM_AUTHOR("name")Set author
Page Setup
ALBUM_PAGES_SIZE(w h)Page dimensions in mm
ALBUM_PAGES_MARGINS(l r t b)Page margins in mm
ALBUM_PAGES_BORDER(w1 w2 w3 w4)Triple line border
ALBUM_PAGES_SPACING(h v)Horizontal / vertical gap
Pages & Content
PAGE_STARTBegin a new page
PAGE_TEXT("FS" size "text")Left-aligned text
PAGE_TEXT_CENTRE("FS" size "text")Centered text
PAGE_TEXT_RIGHT("FS" size "text")Right-aligned text
PAGE_VSPACE(mm)Vertical spacing
PAGE_RULE_HHorizontal rule line
Stamps
STAMP_ADD(w h "desc" "cat" "cond" "acc")Add stamp in row
STAMP_ADD_AT(x y w h "desc" "shape" "bdr" "fill")Stamp at position
STAMP_ADD_IMG(x y w h "img" "desc" "" "")Image at position
STAMP_ADD_TRIANGLE(w h "desc")Triangle stamp
STAMP_ADD_DIAMOND(w h "desc")Diamond stamp
STAMP_ADD_OVAL(w h "desc")Oval stamp
Rows
ROW_START_FS("FS" size gap width)Fixed-space row
ROW_START_ES("FS" size gap width)Equal-space row
ROW_START_JS("FS" size gap width)Justified row
Columns
PAGE_COLUMN_START(n gap)Start n-column layout
PAGE_COLUMN_NEXTNext column
PAGE_COLUMN_STOPEnd column layout
Fonts
ALBUM_DEFINE_FONT("ID" "Name")Define custom font
HN / HB / HI / HSHelvetica variants
TN / TB / TI / TSTimes variants
CN / CB / CI / CSCourier variants
Colors
COLOUR_ALBUM_BORDER("color")Border color
COLOUR_ALBUM_TITLE("color")Title color
COLOUR_STAMP_BORDER("color")Stamp border color
COLOUR_STAMP_BACKGROUND("color")Stamp fill color
Headers & Footers
ALBUM_PAGES_TITLE("FS" size "text")Page title
ALBUM_PAGES_FOOTER_NUMBER("FS" size align "pre" "post")Page numbers
